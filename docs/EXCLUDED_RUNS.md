# Excluded runs

The public artifact set is intentionally restricted to the validated posterior-learning and evidence branches used in the manuscript.

Excluded runs:

- 0113: miscentered restricted-memory MCMC branch, superseded by 0114.
- 0117: confounded failure branch in which solver choice and prior volume were changed simultaneously, making the result scientifically uninterpretable for public reproducibility purposes.

These runs are acknowledged for transparency but are not included as public reproducibility artifacts.
